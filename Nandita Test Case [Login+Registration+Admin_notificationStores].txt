#Retail_User_Login

Validate user with correct credentials i:e Username and Password
-Username: contains username format eg. test@gmail.com, user@yahoo.com
-Password: Check  for below password validators:
1. Password must cotain one lowercase letter.
2. Password must contain one uppercase letter.
3. Password must contain one number.
4. Password must contain one special character [@#$%*/].
5. Password length should be not less than 8 character length.

-Confirm Password: Confirm password should match the password.

Check both the fields are inputed, otherwise unauthorized user
